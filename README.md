# Configure

## GNU Stow

See docs for `GNU Stow` hosted on the official [website](https://www.gnu.org/software/stow/)
or paginate through `man stow`.

TL;DR: run `stow --target=HOME <package>`.

## Core package

```console
stow --verbose=2 --target=$HOME --restow core

```

## Editor package

```console
stow --verbose=2 --target=$HOME --restow editor

```
